The new campus design is ... interesting. 25 buildings, Building A through Building Y, are set up in a grid of 5 blocks by 5 blocks. Each building is from 1 to 5 stories tall. Each row and each column contains exactly one building of each height, from 1 to 5 (like a Sudoku).

From any side, ignoring perspective, taller buildings block the view of shorter buildings. Thus, the 5 story building in a row (or column) is always visible, but a 3 story building is visible only if it is not blocked by a closer 4 or 5 story building. A 1 story building is visible only if it is the closest building, because otherwise it would be blocked.

The outside edges of the diagram below indicate all buildings that are visible when looking directly along the row or column, in the direction of the arrow, but not necessarily in that order. In this example row, from the left, C is blocked by E, and D is blocked by A; but from the right, B blocks the view of all other buildings:
