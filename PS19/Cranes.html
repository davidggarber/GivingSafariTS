New buildings are sprouting up all over! Each building requires a crane and that crane can only work on that building. A crane must be right next to the building it’s working on (no diagonals) but cranes may collide with each other during operation if they’re adjacent (even diagonally) so that must be avoided! Thankfully some helpful surveyors have walked the construction site perimeter and noted how many cranes should be in each row and column.
