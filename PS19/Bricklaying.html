This puzzle is a Futoshiki. Each cell can contain one of the numbers 1, 5, 9, 13, 17, 21. Each row and each column contain each number exactly once. The greater and less than inequalities apply to the numbers.

You should ignore the small italic grey numbers to the left, and the light colour backgrounds. They will be used after you have solved the logic part.

Once you’ve completed the grid, extract a letter from each green shaded squares. To get the letter, sum the cell value with the small grey number in the left column. Then convert that number to a letter using alphabet indexing. You can find the correct order to extract the green squares in the location data shown below – so there is no need to anagram. The final answer is a single 11 letter word.
