To enable a decoder shortcut in the footer, the boilerplate must contain:

        const boiler = `{
          abilities: `{
            decoder: true | code,
         `},
        `}

The decoder will appear in an <iframe>, to the right of the puzzle.

If decoder:true, the link will be to the decrypt.fun home page.

Or specify a specific encoding by name, and the link will go directly to that (in decode-mode).

Available decodings (Case Sensitive!!!):

  • true (no specific decoder)
  • Braille
  • Morse
  • Semaphore
  • PigPen
  • FiveBit
  • Trinary
  • Nautical
  • Rot (e.g. Rotation)
  • Ascii (e.g. Index)
