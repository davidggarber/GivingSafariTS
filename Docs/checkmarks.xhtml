To enable check marks, the boilerplate must contain:

        const boiler = `{
          abilities: `{
            checkMarks: true,
         `},
        `}

Mark any item for cross-off by class='cross-off'.

The item must be a container type. The check itself is achieved by inserting <span class='check' />
.check is a class with a default background-image: ✔️. These can be overridden as a puzzle's needs demand.

Overriding styles

To override which image represents a check or its size:

        #YourPuzzle .check::before `{
            background-image: url('../Icons/Check.png');
            background-size: 30px 27px;
            width: 30px; 
            height: 27px;
        `}

To override its position on the container that was checked:

        #YourPuzzle .check `{
          position: absolute;
          font-size: 16pt;
          left: -0.6em;
          top:-0.5em;
          width: 1em;
        `}
