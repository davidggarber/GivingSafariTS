There are three components to set up.

  1. The stampable elements. Perhaps a container like a div or table cell. Or just an item that gets modified.
  2. A stamp palette, listing all of the possible stamps. This doesn't have to be visible, but does have to exist.
  3. The stamping effects. These can be CSS classes or template contents that get added to containers.

Style-based stamps

Tag each grid cell "stampable".

Skip any that should be locked.

Create styles for each tool.
Tools can specify multiple, separated by spaces.

These classes will be applied to the 'stampable' elements.

There can only be one palette per page.
It must have id='stampPalette'.

Each tool must have class='stampTool'

Each tool must have a unique ID.
These are used by the stamping code to rotate or toggle through stamps.

data-style is a list of classes to apply.

data-unstyle is a list of classes to remove.
If set on the palette, it is invoked each time, prior to applying a new style.
