One of the boilerplate object's fields is 'abilities'.
It can have any combination of the following...

checkMarks Players can check off hints they've used. ✔️
decoder Popup window with handy decoders.
dragDrop Some objects inthe puzzle can be drag-dropped. 👈
hashiBridge Bridges can be drawn between nodes, by Hashi rules. 🌉
highlights Objects/cells can be highlighted, to aid in extraction. 💡
notes Extra text areas where players can take notes to self.
stamping Players can fill in regions by stamping from a palette. ✒️
straightEdge Players can connect-the-dots. 📐
subway Enables code that draws subway-style connectors between input cells.
wordSearch Players can circle words in a grid - optionally with turns. 💊
