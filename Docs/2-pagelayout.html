All fields of the boiler are optional. Below, author and copyright are omitted.

Additional features are appropriate for some puzzles are events. Setting them in boiler will configure everything. 


<html>
  <head>
    <meta charset="UTF-8"></meta>
    <script> var exports = {}; </script>
    <script src="../js/kit24.js"></script>
    <script>
      const boiler = {
        safari: '24',
        title: 'Your puzzle title',

        pageSize: 'letter',
        orientation: 'portrait',
        googleFonts: 'Chewy,Playwrite PE',
        textInput: true,
        type: 'word',
      };
    </script>
    <style>
      #pageBody { font-family: 'Chewy'; font-size: 20pt; }
      .play { font-family: 'Playwrite PE'; }
    </style>
  </head>
  <body id="YourPuzzleTitle">
    <div id="pageBody">
      Fancy <span class='play'>fonts!</span>
    </div>
  </body>
</html>
 










← Set the paper size. Letter is default. Others options: legal, tabloid, ledger, scrapbook.
← Set to either portrait (the default) or landscape. This document page is ledger + landscape.
← Any additional Google fonts to automatically load. Separate with commas. Spaces allowed within names.
← Enables text input functionality, when puzzles contain lots of little entry boxes.
← Place a puzzle-type icon in the top-right corner of the page. Helpful for novice puzzlers to pick a puzzle.
    Examples: word       logic       math       search       code       reassemble
