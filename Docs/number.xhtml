Creates a small input field, sized for a single numeric character.

  • On devices with a virtual keyboard, it should default to numeric input.

    Generated HTML

    Additional attributes and styles both identify the letter-cells of <number>s, and achieve the numeric constraint.

    <span class="letter-cell numeric">
         <input type="text" class="letter-input" pattern="[0-9]*" inputmode="numeric" />
    </span>

    Regional overrides

    Exactly the same optional styles are available. They will each be slightly wider, but otherwise equivalent.
