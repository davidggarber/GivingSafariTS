URL Arguments

trace Lightweight debugging.
Causes builder elements like <for> and the input tags to mirror the half-built states to the console.
debug Heavyweight debugging.
Features can pop up alerts, or color elements.
body-debug Layout debugging.
Causes page layout elements to be rendered with background colors.
iframe Alerts page layout that it is inside an iframe - usually for printing.
Should suppress floating page visual.
print Alerts page layout that it is about to be printed.
Any landscape pages will rotate 90° so that all pages have portrait rotation.
Should also have the same effect as CSS' @media print { } when actually in the printing view.
icon Alerts page layout that we are about to capture a screenshot for purposes of creating icons.
Very similar to print above, with small adjustments for the intended purposes.
restart Overrides any saved page state, and skips the "Reload" vs. "Start over" dialog.
Equivalent to always picking "start over".
restart Always loads any saved page state, skipping the "Reload" vs. "Start over" dialog.
Equivalent to always picking "reload".

On-element debug attributes

<body class='debug'> Layout debugging.
Causes page layout elements to be rendered with background colors.
<anytag … nothrow=''> Suppresses Puzzyl Builder exceptions while constructing this element and its contents.
If an exception happens, the builder will continue to construct what it can.
It will still log any errors to the console.
